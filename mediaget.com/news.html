Патч
Існує спеціальний БЕЗКОШТОВНИЙ патч, який дозволяє вирішити проблему повільної роботи браузера при ввімкненому MediaGet (або будь-якого іншого менеджера завантажень).Патч підтримує: Windows XP SP1(x64)/SP2(x86,x64)/SP3, Vista без SP(x86,x64) і з SP1(x86,x64), 7 beta(x86,x64).Патч дозволяє усунути обмеження на кількість напіввідкритих з’єднань (half-open connections), яке було введено Microsoft в Windows XP SP2 і усунуто в Windows Vista SP2 та наступних версіях. У фінальній версії Windows 7 цієї проблеми вже немає.Щоб завантажити патч, натисніть кнопку знизу.У випадку якої-небудь помилки, можете повернути все, як було. Для цього натисніть "Відновити початковий файл". Встановлення Half-open limit fix:1) Виконайте Half-open limit fix2) Оберіть нову межу кількості одночасних напіввідкритих вихідних з’єднань та натисніть «Внести в tcpip.sys» (для більшості користувачів буде достатня межа, що дорівнює 100)3) Перезавантажте комп’ютер і насолоджуйтеся.Може знадобитися повторний запуск Half-open limit fix після оновлення Windows або встановлення SP, оскільки іноді Microsoft замінює старий файл tcpip.sys на новий.Якщо у вас залишилися запитання - напишіть нам, за допомогою форми зворотнього зв’язку Завантажити патч
