  • Entrez un nom dans la case de recherche

  • Sélectionnez le fichier souhaité parmi les options de recherche

  • Cliquez sur "télécharger" et Choisissez l'emplacement dans lequel vous souhaiter enregistrer votre fichier

  • Une fois le téléchargement terminé, ouvrez le fichier

  • Attendez quelques minutes pour que le film se charge et commencer à regarder en ligne

  • 1

    Le programme possède une barre de recherche qui permet de trouver n'importe quel fichier par son nom. Regarder la vidéo

  • 2

    Sélectionnez le fichier souhaité et cliquez sur télécharger.Nombre d'étoiles depend de la vitesse de téléchargement (Plus d'étoiles signifie que la vitesse de téléchargement trop élevée).

    Pour affiner vos recherches, vous pouvez utiliser les options de recherche avancée. Regarder la vidéo

  • 3

    Une boîte de dialogue s’affiche, dans laquelle vous pouvez sélectionner l'emplacement par défaut où seront enregistrés vos téléchargements ou sélectionner un emplacement différent pour chaque téléchargement.

    Lorsque vous cliquez sur "Détails du Torrent", vous pouvez sélectionner les fichiers à télécharger. Regarder la vidéo

  • 4

    Pour ouvrir un fichier téléchargé, Appuyez sur le bouton droit de la souris et sélectionnez l'onglet "Ouvrir le dossier"..

  • 5

    Vous pouvez consulter Le fichier sélectionné en ligne, sans attendre la fin du téléchargement. Regarder la vidéo

  • 6

    Vous pouvez partager vos fichiers préférées avec vos amis.Il suffit de glisser le fichier téléchargé dans la fenêtre située dans le coin gauche. Regarder la vidéo
