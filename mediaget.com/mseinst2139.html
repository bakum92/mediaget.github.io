Как Добавить Mediaget В Исключения Антивируса Microsoft Security Essentials (Инструкция)
Уважаемые пользователи!
Если вы столкнулись с такими проблемами, как:
- не могу скачать MediaGet (антивирус определяет программу как потенциально опасное ПО, вирус, угрозу, троян и тд.);
- не могу установить MediaGet (вместо окна установщика - "белый лист");
- не могу скачать файл (не добавляется в "загрузки", нет скорости при наличии раздающих);
вам необходимо добавить файл mediaget.exe или папку MediaGet2 в исключения вашего антивируса. Ниже приведена примерная инструкция для антивируса Microsoft Security Essentials.
1. Нажмите правой клавишей мыши на значок антивируса Microsoft Security Essentials в трее и выберите опцию "Открыть";
2. Перейдите во вкладку "Параметры" и в блоке слева выберите пункт "Исключенные файлы и расположения";
3. Нажмите на кнопку "Обзор", выберите папку "MediaGet2", расположение которой указано ниже и нажмите "Ок", затем "Добавить";
* для Windows XP: C:\Documents and Settings\ИмяВашегоКомпьютера\Local Settings\Application Data\MediaGet2
* для Windows 7: C:\Users\ИмяВашегоКомпьютера\AppData\Local\MediaGet2
4. Для завершения настройки нажмите на кнопку "Сохранить изменения".
