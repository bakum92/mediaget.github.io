Patch
Há um patch ("remendo") gratuito que pode resolver o problema de um navegador lento quando MediaGet (ou qualquer outro cliente download manager) estiver activo.

O patch é compatível com Windows XP SP1 (x64) / SP2 (x86, x64) / SP3, Vista sem SP (x86 , x64), Vista com SP1 (x86, x64), 7 beta (x86, x64).

O patch permite remover a restrição no concorrente de conexões semi-aberto (half-open connections) de saída TCP, que foi introduzida pela Microsoft no Windows XP SP2 e, em seguida, removido no Windows Vista SP2 e versões posteriores. A versão final do Windows 7 não tem esses problemas.

Para mais informações, por favor, verifica na pagina inicial.

Para baixar o Patch, por favor clique no botão abaixo.


Em caso de qualquer erro você pode restaurar as configurações padrão. Pressione "Recuperar o arquivo de origem" para isso.

Instalação de Half-open limit fix (restrição semi-aberto e correção):
1) Executar Half-open limit fix

2) Selecione um novo Half-open limit fix de saída TCP e pressione ‘Importar para tcpip.sys’ (100 restrições de conexão é suficiente para a maioria dos usuários).

3) Reinicie o computador e desfruta do bom desempenho.
É provavel que terá que reiniciar o Half-open limit fix após a atualização do Windows ou instalar o Service Pack Microsoft como, por vezes, substitui o arquivo tcpip.sys antigo por um novo.
